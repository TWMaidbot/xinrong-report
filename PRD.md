## 1. 產品概述

### 1.1 背景
針對新榮工廠（XinRong Manufacturing）開發的數位化報工系統，旨在解決紙本或舊系統操作繁瑣的問題。透過行動優先的網頁介面，結合現場 QR Code 掃描，達成「秒級報工」與「即時戰情」的目標。

### 1.2 核心目標
1.  **極速報工**：操作流程 < 10 秒（掃碼 -> 自動帶入機台 -> 選擇工項 -> 提交）。
2.  **即時戰情**：大螢幕與手機同步顯示產量、機台進度與員工績效。
3.  **零門檻**：TV 戰情模式免登入，一鍵投射到工廠電視。
4.  **數據安全**：防止惡意刪改，並支援數位化保存與匯出。

## 2. 使用者角色與權限

### 2.1 角色定義
| 角色 | 說明 | 權限內容 |
|------|------|----------|
| **一般員工 (Worker)** | 機台操作員 | 報工、查看個人紀錄 |

| **戰情看板 (TV)** | 公用顯示器 | 僅供讀取的唯讀模式（免登入） |

## 3. 系統功能

### 3.1 報工模組
- **QR Code 掃描**：整合相機功能，直接掃描機台 QR Code 自動帶入機台編號。
- **機台記憶 (Persistence)**：系統自動記憶上次使用的機台，減少重複操作。
- **智慧分類**：依據選定的機台（如剪台），自動篩選對應的工作類型（如原木、餘料）。
- **共同作業**：支援多選協作者，產量共享。
- **快速數量**：提供 +1, +10, -1 等快捷按鈕或直接輸入。

### 3.2 戰情看板 (Dashboard) & TV 模式
- **專屬 TV 模式**：
  - 網址參數 `?mode=tv` 啟動。
  - **免登入**：適合電視瀏覽器首頁設定。
  - **1080p 優化**：去除選單、放大字體、自動置中排版。
  - **自動刷新**：每 60 秒自動更新數據。
  - **即時時間**：標題顯示當前日期與星期。
- **視覺化指標**：
  - 各類別今日產量統計。
  - 各機台即時進度條 (紅/綠/黃 燈號)。
  - **員工績效排行榜**：Top 3 頒發獎盃圖示 (🏆, 🥈, 🥉)。

### 3.3 管理模組
- **使用者管理**：新增、編輯員工資料與權限。
- **機台管理**：新增、編輯機台，並可 **一鍵生成/列印機台 QR Code**。
- **報工紀錄**：具備日期區間、員工、機台等多維度篩選功能。
- **報表匯出**：支援 CSV 匯出（Excel 相容格式）。

## 4. 技術架構

### 4.1 技術選型
| 層級 | 技術 | 說明 |
|------|------|------|
| **前端** | Vanilla JS (ES6+) | 輕量化，無須編譯流程 |
| **樣式** | CSS3 (Flex/Grid) | 原生響應式設計，無 CSS 框架依賴 |
| **後端** | Firebase SDK 9 | Serverless 架構 |
| **資料庫** | Cloud Firestore | NoSQL 即時資料庫 |
| **部署** | Firebase Hosting | 全球 CDN 快速分發 |
| **掃描** | html5-qrcode | 跨平台 QR Code 掃描庫 |

### 4.2 資料安全 (Security Rules)
- **讀取 (Read)**：全域開放 (為了支援 TV 模式免登入)。
- **寫入 (Create/Update)**：必須包含必要欄位 (worker, machine, quantity) 才允許寫入。
- **刪除 (Delete)**：**全面禁止** (防止資料被惡意或誤刪)，僅允許新增與修改。
